## ✨ Core Features

### 📊 Advanced Engine Insights (GitHub)
- **Real-time Synchronization**: Live stream of your GitHub contribution data.
- **Language Mastery**: Visual breakdown of your technology stack proficiency.
- **Commit Velocity**: Heatmap visualization of your coding consistency over the last 30 days.
- **Pulse Feed**: Instant tracking of recent pushes, pull requests, and repository creations.

### 🧩 Algorithm Proficiency (LeetCode)
- **Problem Solving Matrix**: Tracking your solved problems and global ranking.
- **Success Rate**: Monitoring your algorithmic growth and consistency.

### 🎯 Strategic Roadmap & Goals
- **Dynamic Milestones**: Adaptive roadmap that evolves with your developer level.
- **Goal Center**: Set and track weekly technical objectives with progress visualization.
- **Achievement System**: Unlock premium badges as you reach significant milestones.

### 📱 Responsive & Premium UI
- **Glassmorphic Design**: A state-of-the-art interface with smooth transitions and blur effects.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile devices with a toggleable sidebar.
- **Dark Mode Native**: Designed for late-night coding sessions with high-contrast, easy-on-the-eyes colors.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vite, Vanilla CSS (Premium Glassmorphism)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Integrations**: GitHub GraphQL & REST APIs, LeetCode Public API

## 🚀 Quick Start

### Prerequisites
- Node.js (v16+)
- MongoDB instance

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/skill-tracker.git
   cd skill-tracker
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Create a .env file with your MONGO_URI and JWT_SECRET
   npm start
   ```

3. **Frontend Setup**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
